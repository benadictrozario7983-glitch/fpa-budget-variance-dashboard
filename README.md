📊 Project Overview
This repository contains a comprehensive Financial Planning and Analysis (FP&A) dashboard designed to monitor budgetary control and organizational performance. The project transforms raw transactional data into actionable insights, allowing stakeholders to identify spending trends, budget overruns, and departmental efficiencies.

The core objective is to automate the comparison between Actual vs. Budgeted figures to facilitate data-driven decision-making.

🛠️ Key Features & Business Logic
Budgetary Control: Automated tracking of department-wise spending against pre-allocated budgets.

Variance Analysis: Calculation of absolute and percentage variances to highlight favorable and unfavorable deviations.

Performance Management: Visual indicators for Key Performance Indicators (KPIs) to assess fiscal health.

Dynamic Interactivity: Users can filter data by Month, Department, or Expense Category using integrated Slicers.

🧪 Advanced Excel Toolkit
This project leverages advanced Excel functionality to ensure data integrity and dynamic reporting:

Logic & Lookup:

SUMIFS: Used for multi-criteria data aggregation across different timeframes and departments.

XLOOKUP: Employed for robust data mapping and fetching metadata from reference tables.

Data Summarization:

Pivot Tables: The engine behind the dashboard, used to categorize and summarize thousands of rows of financial data.

Visual Analytics:

Conditional Formatting (Data Bars): Integrated directly into tables to provide instant visual context for variance magnitude.

Interactive Charts: Dynamic bar and trend lines that update automatically based on user selection.

Slicers: Provided for a "point-and-click" user experience, making the report accessible to non-technical stakeholders.

📂 Repository Structure

Data/: (Optional) Sample raw datasets used for the analysis.

Screenshots/: Visual previews of the dashboard for quick viewing in GitHub.

🚀 How to Use
Clone the Repo: Download the .xlsx file to your local machine.

Enable Macros/Content: If prompted, enable content to allow the Pivot Tables and Slicers to function.

Interact: Use the Slicers on the left/top of the dashboard to filter through different fiscal periods or departments.

Update: To use your own data, paste new transactions into the Data_Input sheet and click Data > Refresh All.

📈 Insights Provided
Top 5 Expense Drivers: Identification of which categories are consuming the most budget.

Burn Rate: Monthly visualization of how quickly the annual budget is being utilized.

Departmental Accountability: A clear view of which managers are staying within their fiscal limits.
